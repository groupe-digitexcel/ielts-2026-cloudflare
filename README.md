# IELTS 2026 Cloudflare

AI-powered IELTS 2026 Exam Simulator with:
- React + Vite frontend
- Cloudflare Pages
- Cloudflare Worker backend
- D1 database
- KV caching
- R2 reference storage
- Admin uploads
- AI-style generated exam variants
- Automatic evaluation
- Local payment + Visa placeholder integration

## URLs
- Frontend: https://groupe-digitexcel.pages.dev
- Backend: https://groupe-digitexcel.workers.dev
- API: https://groupe-digitexcel.workers.dev/api

## Commands
- npm install
- npm run db:apply
- npm run deploy:worker
- npm run deploy:pages
